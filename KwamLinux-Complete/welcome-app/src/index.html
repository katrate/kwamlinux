Quick Start
Follow these steps to get the most out of Came Linux.
✓
Came Linux is installed
You are running Came Linux 1.0.0. The base system, KDE Plasma, and security tools are ready.
2
Update your system
Always start by bringing your system up to date.
kwamlinux update
3
Install security tools
Get the full Came security toolkit — nmap, Metasploit, Wireshark and more.
kwamlinux security install
4
Harden your system
Enable UFW firewall, fail2ban, AppArmor and SSH hardening in one command.
kwamlinux harden
5
Install an AUR package
Access 80,000+ packages from the Arch User Repository.
kwamlinux install --aur <package>
6
Run a Windows app
Install and run Windows software using Wine and Bottles.
kwamlinux windows install

Applications
Came Linux supports native Linux apps and Windows apps via Wine.
🐧
Linux Apps (pacman / AUR)
80,000+ packages available. Install anything from the official repos or AUR.
kwamlinux search <app>
🍷
Windows Apps (Wine)
Run .exe applications using Wine with Bottles for easy management.
kwamlinux windows install
🎮
Gaming (Proton)
Play Windows games on Came Linux via Steam + Proton or Lutris.
kwamlinux install lutris
🐳
Containers (Docker)
Run any app in isolated containers with Docker and Docker Compose.
kwamlinux install docker
